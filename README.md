# CardsBoard 🃏

[![Open App](https://img.shields.io/badge/Open%20App-▶%20Launch-green?style=for-the-badge)](https://hexerre.github.io/CardsBoard/)

A virtual card table for cards games — runs entirely in your browser, 
no install, no account, no server.

---

## What you can do

**Deck**
- Draw one card or N cards at once
- Shuffle with animation
- Browse and search the full deck by suit or value
- Configure deck size (1×, 2×, 3×) or add Jokers
- Remove entire suits from the deck
- Infinite deck mode (cards never run out)

**Cards**
- Click to flip face up / face down
- Right-click for a context menu: send to table, take in hand, 
  move to top/bottom of deck, move to any custom zone
- Drag & drop between all zones (mouse and touch/mobile)
- Rotate 90° at a time
- Add/remove counters on individual cards

**Zones**
- 🟩 **Table** — shared play area, supports snap-to-grid
- 🤚 **Hand** — tabbed, one per player; sort by value or suit
- 📦 **3 custom zones** — renameable (e.g. Oracle, Discard, Reserve)
- 🗑️ **Discard pile** — dedicated sidebar zone

**Players**
- Add multiple player tabs with custom names
- Per-player notes
- Score tracker with +/- controls

**Extra**
- Dice roller (D4 to D100)
- Activity log with timestamps
- Auto-save via localStorage — your table persists across sessions
